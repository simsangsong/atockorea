## Admin v1 → v2 promotion: v2 module becomes `/admin/products`

Promotes the Phase 2 admin product editor (three-pane: list + editor + live preview) from `/admin/products/v2` to the root `/admin/products` route, and removes the 2881-LOC v1 page in the process.

### Why this PR

Agreed on 2026-05-22 but never shipped — the v2 module was committed under `app/admin/products/v2/` but the v1 single-file page stayed at the root route. Two side effects:

1. **Production still served v1** — sidebar still pointed at `/admin/products/v2`, but anyone who typed `/admin/products` (or had it bookmarked) hit the legacy 2881-LOC monolith.
2. **Two routes for the same feature** — the v2 path also lived under `/admin/products/v2`, with a "구버전" breadcrumb label on the root.

This PR settles the migration: v2 becomes the canonical `/admin/products`, the v2 subdirectory disappears, and every internal link points at the root.
