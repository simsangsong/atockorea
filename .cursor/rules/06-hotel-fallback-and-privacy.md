# Hotel Search Fallback & Privacy

- Hotel search must support:
  1. autocomplete
  2. landmark chip fallback
  3. manual pin fallback
  4. manual review flag if still unresolved

- Analytics must never store:
  - raw hotel name
  - exact coordinates
  - user-precise geolocation

- Analytics may store:
  - pickupAreaLabel
  - surchargeTier
  - tourType
  - booking state
  - conversion step

- Customer-facing language must remain human and non-technical.
