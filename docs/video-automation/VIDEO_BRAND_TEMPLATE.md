# Video Brand Template

## Baseline

- Format: vertical `9:16`
- Resolution: `1080x1920`
- Duration: 55-65 seconds, Phase 1 fixed at 60 seconds
- Brand: ATOCKOREA Smart Guide
- Captions: WebVTT and SRT generated for `en`, `zh-Hant`, `ja`, `es`
- Narration: one narration text/audio placeholder per target language

## Scene Template

| Time | Template | Purpose |
| --- | --- | --- |
| 0-3s | Hero intro | Hook and POI name |
| 3-12s | POI title | Core identity |
| 12-28s | Culture card | Verified background |
| 28-43s | Must-see highlight | Main viewing point |
| 43-53s | Visit tip | Practical or experience tip |
| 53-60s | Smart Guide CTA | App handoff |

## Layout Rules

- Keep subtitle text within mobile safe areas.
- Do not bake frequently changing facts such as current operating hours or prices into the video unless verified for the publication date.
- Prefer app real-time cards for operating hours, fees, holidays, and transport timing.
- Mark AI-generated narration in metadata.
- Generated imagery must be identified as generated and cannot impersonate real footage.
