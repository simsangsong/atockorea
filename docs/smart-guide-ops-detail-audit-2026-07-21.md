## 3. 플랜

### TIER A — 실전 임팩트 (착수 순서 A0→A2→A4→A1→A3→A5→A6)

**A0 — 도착 원버튼 번들 (최우선, 사용자 직접 요구 2026-07-21)**
버스 안전요원/쏠라티 기사 단독 리드용. 관광지 도착 → 콕핏 [도착] → 시트 → [전원 발송].
- **2계층 확정(사용자, 2026-07-21):** 스톱별 고정=사전 설정(디테일 카드·화장실 핀·인솔/자유·관람노선 노트·티켓 여부) / **당일 변수=현장 입력 2개뿐(집합시간·주차핀)**.
- 주차핀: 시트 열 때 GPS 자동 캡처(방금 주차한 곳), 지도 미니뷰 확인, 드래그 보정.
- 집합시간: 상대 칩(+30/40/60/90) + 시각 직접 입력. **기본값 없음 — 반드시 1탭**(어제 시간 오발송 방지). "집합 없이 보내기"(짧은 포토스톱) 옵션.
- 스톱 프로필: 시트에서 첫 설정 → poi_key 단위 저장(sticky defaults) → 어드민 에디터 없이 v1 완결. 노트는 발송 시 번역(T2-2 패턴)+캐시.
- 발사 체인(서버 1라우트): ①meeting_notice(기존 카운트다운·rally 사다리 무변경 동작, T2-1 핀 결합) ②parking_pin ③arrival_bundle 카드(스팟 사진 헤더+화장실 지도+주차+인솔 배지+노선+티켓 줄). 공유 투어면 투어일 전 예약 팬아웃(T2-3 재사용). 푸시 1건.
- 현장 플로우 = **3탭**(도착→시간 칩→발송).

**A1 — 모닝 브리핑 카드**: 투어 시작 시 자동/원탭 발사. 조인=주의사항·일정링크·점심·복귀 / 프라이빗=차터 기준시간(제주9/부산8)·오버타임 30k/h·정방향 규칙. 5로케일 zero-LLM. A0 번들 컴포저 패턴 재사용.
**A2 — 다음 스톱 ETA 배선(G1)**: matrix 읽기+haversine → "20km·약 23분". A0 번들 꼬리줄 결합.
**A3 — 드랍포인트 변경+픽업요청 응답 루프(N3+G8)**: 손님 버튼→지도핀(±사진)→콕핏 알람→숫자 프리셋 "N분 후 도착" 원탭 회신.
**A4 — 당일 이벤트 상태(G4)**: per-day 진행 O/X 운영자 토글→손님 카드 인용. 해녀쇼 우선, 일반화 모델.
**A5 — 앱 사용 설명서(N2)**: 조인/프라이빗 별도, 5로케일, 룸 진입 1회+상시 메뉴. 역할분리 문구.
