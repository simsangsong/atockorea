## Steps to Add

1. Open your Supabase project
2. Go to SQL Editor
3. Copy the contents of `supabase/insert-jeju-winter-tour.sql`
4. Paste into the SQL Editor
5. Click "Run" or press `Ctrl+Enter`
6. Verify the tour was created successfully
