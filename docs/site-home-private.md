# 사이트 외부 차단 (localhost 만 전체 UI)

## 동작

`SITE_HOME_PUBLIC`이 **설정되지 않았거나** `false`인 경우:

| 접속 Host | `/`, `/tours`, `/tour/...`, `/mypage` 등 **UI 경로 전부** |
|-----------|-------------------------------------------------------------|
| **localhost**, **127.0.0.1**, **::1** (`:3000` 등) | **정상 앱** |
| **그 외** (atockorea.com, 프리뷰 URL, 공인 IP, LAN IP 등) | **비공개 안내** (`/home-private`로 rewrite, URL 표시줄은 요청 경로 유지, `noindex`) |

- **`/api/*`**, **`/_next/*`**, **확장자 있는 정적 파일**은 그대로 통과 (rewrite 없음).
- `next dev -H 0.0.0.0` 후 **휴대폰에서 PC IP로 접속**하면 막힘 → **`http://localhost:3000`** 또는 **`http://127.0.0.1:3000`** 사용.

## 공개 오픈 시

환경 변수 **`SITE_HOME_PUBLIC=true`** (또는 `1`) + 재배포.  
다시 막으면 변수 제거 또는 `false` 후 재배포.
