# 메인 페이지 — 외부 차단 (localhost 만 실제 홈)

## 동작

| 접속 Host | `/`, `/ko`, … (로케일 **루트**만) |
|-----------|-----------------------------------|
| **localhost**, **127.0.0.1**, **::1** (`:3000` 등) | **실제 홈** |
| **그 외** (배포 도메인, 공인 IP, `192.168.x` 로 접속 등) | **비공개 안내** (같은 URL에 rewrite, `noindex`) |

- `/tours`, `/tour/...`, API 등 **다른 경로는 막지 않음**.
- `next dev -H 0.0.0.0` 후 **휴대폰에서 PC IP로 접속**하면 홈은 막힘 → **`http://localhost:3000`** 또는 **`http://127.0.0.1:3000`** 사용.

## 잠깐 전체 공개 (배포 후)

환경 변수 **`SITE_HOME_PUBLIC=true`** + 재배포.  
다시 막으면 변수 제거 또는 `false` 후 재배포.
