# 조인투어 파일럿: Grand Highlights Jeju 프리셋 + 현장 자동화 (2026-07-22)

**이 문서가 파일럿 트랙의 SoT.** 상위: `docs/join-tour-ticketless-rich-itinerary-master-plan-2026-07-22.md`(J0~J4 완료).
파일럿 투어 = `jeju-grand-highlights-loop` (price_type **person** = 공유 조인투어, 번들 당일 전예약 팬아웃).

## §A 사용자 바인딩 결정 (2026-07-22)

1. **주차핀·집합시간 = 매일 다름 → 기사 현장 수동.** 콕핏에서 [현재 위치로 핀] 버튼 + 시계 다이얼로 집합시간. 프리셋에 절대 고정하지 않음.
2. **나머지(인솔/티켓·가격/관람순서/이벤트) = 프리셋 + 지오펜스 자동.** 차량이 스톱 반경에 들어오면 프리셋 내용이 손님 대화창에 자동 카드 + 알림(푸시). 기사는 도착 후 핀·시간만 만짐.
3. **아침 픽업 = 손님 지도에 [차량 위치 보기].** 픽업 대기 중 버스가 어디쯤인지 실시간 확인.

## §C WBS

### G1 — 도착 프리셋 6종 시딩 (데이터, 즉시)
`tour_poi_arrival_profiles` upsert: follow_mode='free'(조인투어 자유관람 기본), ticket_required+**ticket_krw**(주상절리·정방 2,000 / 성산 5,000 / 만장굴 4,000), route_note 한국어 초안+5로케일 i18n 동봉(R-6 폴백 방지), event_label '해녀 물질 공연 14:00'은 해녀공연 스톱(`jeju_haenyeo_museum`)에. meeting_point는 **null 유지**(§A-1: 기본 "차량 앞", 당일 핀이 실좌표).

### G1.5 — 매표소·화장실 핀 (데이터+검수)
유료 4곳 매표소 좌표 조사 → `ticket_booth` curated+verified 등록(티켓ON 번들에 지도 자동 첨부, J0 배선 완료). 6개 POI 한정 화장실 핀 수집·검수.

### G2 — 차량 지오펜스 자동 번들 (코드, 핵심)
- **콕핏 GPS 워처 1개 신설** = G4와 공유: `watchPosition` → ①'location' 브로드캐스트(role driver/guide = 차량 위치) ②`stepSpotWatch`(기존 스테퍼, 스케줄 스톱+반경) → 도착 판정 시 **arrival-bundle 자동 POST**(meetingTime null, profile 패치 없음 → 서버 스티키 프리셋이 그대로 탑승, 핀 = 현재 좌표).
- **멱등**: 클라 스톱당 1회 가드 + 서버 `tour_room_events` subject_key(`auto_bundle:<poi>:<date>`) 디듀프 — 재접속·리렌더에도 스톱당 하루 1번.
- **푸시 확장**: 자동 번들은 meetingTime이 없어도 푸시(도착 안내 자체가 rally-critical). 태그 디듀프 유지.
- **알림 UX**: 카드는 기존 arrival_bundle 렌더 그대로(팝업=채팅 카드+미확인 배지), 푸시가 시스템 알람.
- **1장 불변식(P-D8)**: 공유투어에서 차량 자동 번들 발사 후 동일 poi 손님측 지오펜스 spot_arrival은 서버에서 억제(최근 번들 존재 시 스킵) — 이중 카드 방지.
- 콕핏 토글: [자동 도착 안내 ON/OFF](기본 ON) — 오작동 시 기사가 끄고 수동 [도착]으로 폴백.

### G3 — 동영상 4종 (사진 3장+ 스톱만)
1100고지·주상절리·정방·성산 4언어 `video:produce --tts=openai`(≈$0.2) → `video:upload` → `/admin/poi-videos` 승인 대기(사람 게이트). 해녀·만장굴은 사진 보강 후.

### G4 — 손님 [차량 위치 보기] (코드)
- 콕핏 워처(G2)가 이미 차량 위치를 브로드캐스트 → RoomMapTab에 **차량 마커**(driver/guide role, 버스 아이콘) + [차량 위치 보기] 버튼(탭 시 차량 좌표로 지도 포커스).
- 홈 탭 픽업 대기 상태(집합 전 아침)에 같은 버튼 노출 → 지도 탭 딥링크.
- 위치 데이터는 기존 R-17 퍼지 정책 그대로(별도 저장 없음, 브로드캐스트만이면 저장 이슈 없음 — 마지막 프레임만 채널 스냅샷).

### G5 — 시뮬 리허설 + 실기기
sim-tour-day(이 투어) → 콕핏 워처에 모의 좌표 주입 → 자동 번들 발사·이중카드 억제·푸시·차량 마커 확인 → 스크린샷 보고. 실기기(운행 중 GPS·백그라운드 제약)는 사람 게이트.
